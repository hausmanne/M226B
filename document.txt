1. Expliquez ce qu’est une méthode virtuelle, une méthode virtuelle pure et une méthode non-virtuelle.
	- méthode virtuelle : c'est une méthode définie dans une classe qui est destinée à être redéfinie dans les classes qui en héritent.
	- méthode virtuelle pure : c'est une méthode qui est déclarée mais non définie dans une classe. Elle est définie dans une des classes dérivées de celle-ci.
	- méthode non-virtuelle : c'est une méthode qui est déclarer et implémenter dans une classe.

2. En Java, comment indique-t-on qu’une méthode est (a) virtuelle (b) virtuelle pure (c) non-virtuelle ?
	a) public abstract
	b) protected abstract
	c) public

3. Expliquez avec vos propres mots la manière dont on utilise le mot clé "super" dans le constructeur de la classe dérivée et la raison pour laquelle on procède ainsi.
On l'utilise dans un constructeur d'une classe pour appeler le constructeur du parent de cette classe. Pour initialiser les variables membres du parent qu'une seule fois.
